# 🎬 CineApp

Aplicación web para buscar y explorar películas, desarrollada con Vue 3 + Vite + Tailwind CSS.

## Tecnologías

- Vue 3
- Vue Router
- Tailwind CSS v4
- TMDB API

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:
   pnpm install
3. Crear el archivo `.env` en la raíz del proyecto con el siguiente contenido:

4. Correr el proyecto:
   pnpm dev

## Funcionalidades

- Listado de películas populares
- Búsqueda por título
- Filtro por género
- Detalle de película
- Agregar/quitar favoritos (localStorage)
